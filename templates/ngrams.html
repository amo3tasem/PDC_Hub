N-grams Tool!

This tool identifies and ranks the most commonly occurring key-terms in a dataset.

It does this by taking in any dataset in csv or xlsx format with a Text Column, discards common terms (Arabic and English stopwords) in the dataset, and then calculates and ranks the frequencies of Unigrams, Bigrams and Trigrams based on your choices.
Input File
Browse for the desired dataset
Text Column
The text column to analyze (case sensitive)
N-grams

Unigrams is a single word e.g.: (tea, hair, ...)

Bigrams is a combination of two words e.g.: (green tea, hair care, ...)

Trigrams is a combination of three words e.g.: (green tea benefits, hair care solutions, ...)

Output File
Output file desired name
